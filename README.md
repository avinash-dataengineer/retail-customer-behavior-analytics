# Retail Customer Behavior Analytics

## Project Overview

This is an End-to-End Data Analytics Project developed using Python, SQL, and Power BI.

The objective of this project is to analyze customer shopping behavior, identify business trends, and generate actionable insights that help improve business performance.

---

## Business Problem

Retail businesses generate huge amounts of customer transaction data every day. This project focuses on analyzing customer shopping behavior and converting raw data into meaningful business insights.
